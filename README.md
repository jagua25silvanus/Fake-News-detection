# IMD Climate Fake News Detector
A climate fake news detector using IMD annual reports (2008-2024).

## Features
- Checks climate claims against IMD reports
- Uses live weather data from Open-Meteo API
- Fine-tuned SBERT + DeBERTa models
- Groq LLM for explanation

## Models
- SBERT: sooriyajs/imd-sbert-finetuned
- DeBERTa: sooriyajs/imd-deberta-finetuned
